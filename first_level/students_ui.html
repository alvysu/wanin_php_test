📚 學生資料 API 操作

後端檔案：students.php

GET：

  • 查詢全部學生
  • 查詢 ID=1 的學生

POST / PUT / DELETE：

請使用 Postman / curl 操作：

新增學生（POST）：
POST https://wanin-php-test.onrender.com/first_level/students.php
Body (x-www-form-urlencoded): name=Jack&score=77

更新學生（PUT）：
PUT https://wanin-php-test.onrender.com/first_level/students.php?id=1
Body (x-www-form-urlencoded): score=95

刪除學生（DELETE）：
DELETE https://wanin-php-test.onrender.com/first_level/students.php?id=2

🔧 可以搭配 Postman 測試上述 API。
